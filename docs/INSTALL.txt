vindula.agendacorporativa Installation
--------------------------------------

To install vindula.agendacorporativa using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``vindula.agendacorporativa`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        vindula.agendacorporativa
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
